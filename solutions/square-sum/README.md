# 🚀 Codewars: Square Sum

Clean and optimal solutions in both Java and Python for the **"Square Sum"** challenge on Codewars.

---

## 📝 Problem Description

The goal is to complete the square sum function so that it squares each number passed into it and then sums the results together.

For example, for `[1, 2, 2]` it should return `9` because $1^2 + 2^2 + 2^2 = 9$.

- **Platform:** Codewars
- **Difficulty:** 8 kyu (Easy)
- **Topics:** Arrays, Loops, Math

---

## 🛠️ Complexity Analysis

- **Time Complexity:** $O(N)$ — Where $N$ is the number of elements in the array/list. We iterate through the collection exactly once.
- **Space Complexity:** $O(1)$ — Constant space. We only use a single variable (`total`) to keep track of the cumulative sum, regardless of the input size.
